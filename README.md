# Snake Game

Desktop Snake game built with Python + pygame.

## Run from source

```powershell
python snake.py
```

## Controls

- Arrow keys: Move
- 1/2/3: Change level
- P: Pause
- R: Restart
- H: Help
- D: Toggle debug line
- Esc: Quit

## Windows build

Use `snake-share.zip` from this repo for a ready-to-run Windows executable.
